📖 ABOUT THE APPLICATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PHARMACY APP is a modern, feature-rich e-commerce platform specifically designed
for online pharmacy services. Built on enterprise-level technology stack, it provides
a seamless experience for customers to order medicines, manage prescriptions, and
for pharmacists/administrators to manage inventory and fulfill orders.

📊 APP SPECIFICATIONS:
├─ Backend: Django REST Framework + PostgreSQL
├─ Frontend: React 19 + Vite + Framer Motion (3D animations)
├─ Search: Meilisearch (Fast, typo-tolerant search)
├─ Payments: Stripe + Razorpay (Multiple payment gateways)
├─ Authentication: Django JWT + Firebase Phone Auth
├─ Real-time: Redis + WebSockets (Celery task queue)
├─ Email: SendGrid integration (production-ready)
├─ Hosting Ready: Docker containerized, cloud-deployable
└─ Target Market: India (Supports INR, local payment methods)
